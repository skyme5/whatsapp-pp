Steps to Install the Extension

1. Open this url in Chrome: "chrome://extensions/"
2. Enable "Developer mode" by clicking toggle switch at top-right corner
3. Click "Load Unpacked" and choose the folder this extension is extracted to.
4. The extension will be installed.
5. Chrome will ask for permissin to download multiple files, allow the permission to download the pictures.
6. The images are downloaded only once when it detects a change. The filename will contain phone number of the person image belongs to and the timestamp it was downloaded.
